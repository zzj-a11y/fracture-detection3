                         第一步：准备材料
========================================================================

就像做饭前要洗菜切菜一样，我们先准备好这些东西：

您需要准备：
1. 一台能上网的电脑（就像能看新闻的那种）
2. 一个电子邮箱（用来注册账号，就像注册微信需要的）
3. 大约30分钟时间（就像煮一锅饭的时间）

准备好了吗？我们开始吧！

========================================================================
                       第二步：注册免费账号
========================================================================

就像注册微信一样简单，我们分几步走：

1. 打开电脑，打开浏览器
   （浏览器就是您平时看新闻的那个程序，比如"百度"打开的窗口）

2. 在最上面的地址栏输入：www.render.com
   （地址栏就是那个长长的白色框，像这样：┌─────────────────┐ ）
                                                    www.render.com
                                                 └─────────────────┘

3. 找到右上角的"Sign Up"按钮（就是"注册"的意思）
   [图片描述：这是一个蓝色按钮，在网页右上角]

4. 点击"Sign Up"按钮后，选择"Sign up with GitHub"
   （GitHub就像一个免费的"云盘"，可以存我们的程序）

5. 如果没有GitHub账号，就注册一个：
   - 输入您的电子邮箱
   - 设置一个密码（记在小本子上，别忘记了！）
   - 按照提示一步一步完成

小贴士：如果不会注册，可以请家里的年轻人帮忙，就像请他们帮忙注册微信一样。

========================================================================
                       第三步：上传程序文件
========================================================================

就像把照片上传到微信朋友圈一样简单：

1. 打开这个网址：github.com
   （还是在浏览器的地址栏里输入）

2. 登录您刚才注册的GitHub账号

3. 点击右上角的"+"号，选择"New repository"
   （repository就是"仓库"的意思，就像一个云文件夹）

4. 给仓库起个名字：fracture-detection
   （其他选项都不用改，就像填快递单只填名字一样）

5. 点击绿色的"Create repository"按钮

6. 现在您有了一个空的"云文件夹"，我们需要把文件放进去：

   a. 在电脑上找到这个文件夹：
      C:\Users\25381\Desktop\zzj\deployment
      （就像在电脑里找照片一样找这个文件夹）

   b. 打开这个文件夹，把里面的所有文件都选中
      （就像在微信里选多张照片一样）

   c. 把这些文件拖到GitHub的网页里
      （就像把照片从手机拖到电脑一样）

   d. 点击"Commit changes"按钮（就是"保存更改"的意思）

恭喜！您已经把程序文件传到网上了，就像把照片传到微信一样成功！

========================================================================
                       第四步：让程序运行起来
========================================================================

就像在手机上下载APP一样简单：

1. 回到Render.com网站（刚才注册的那个）

2. 点击蓝色的"New +"按钮
   [图片描述：这是一个大大的蓝色加号按钮]

3. 选择"Web Service"（网页服务的意思）

4. 连接您的GitHub账号：
   - 点击"Connect account"
   - 选择刚才创建的fracture-detection仓库
   （就像在微信里选择从哪个相册上传照片一样）

5. 填写信息（就像填快递单）：
   - Name（名字）：fracture-detection（就用这个）
   - 其他选项都不用改，就像默认设置一样

6. 点击蓝色的"Create Web Service"按钮

现在就像按了电饭煲的"煮饭"按钮，等它自己煮好！

等待时间：大约5-10分钟
（这个时间您可以：喝杯茶、站起来走走、看看窗外的风景）

您会看到屏幕上有很多字在滚动，这是正常的。
就像电饭煲煮饭时会有声音一样，电脑也在努力工作。

等到出现"Deploy successful"这几个字，就表示煮好了！

========================================================================
                       第五步：使用骨折检测系统
========================================================================

就像用微信聊天一样简单：

1. 等Render显示"Deploy successful"（部署成功）

2. 您会看到一个网址，比如：https://fracture-detection.onrender.com
   （这个就是您的骨折检测系统的"门牌号"）

3. 点击这个网址，或者复制到浏览器里打开

4. 打开后的页面就像这样：

   ┌─────────────────────────────────────┐
   │     儿童手腕隐匿性骨折检测系统        │
   │                                     │
   │      [选择图片] 按钮                 │
   │                                     │
   │      [开始分析] 按钮                 │
   │                                     │
   │      结果显示在这里                 │
   └─────────────────────────────────────┘

5. 使用方法：
   a. 点击"选择图片"按钮
   b. 选择孩子的X光片照片（要jpg或png格式，就像微信能发的照片格式）
   c. 点击"开始分析"按钮
   d. 等待几秒钟（就像微信发照片需要时间一样）
   e. 看结果：系统会告诉您有没有发现骨折

就像用微信发照片给朋友一样简单！

问题1：网站打不开怎么办？
答：就像电视没信号一样，等几分钟再试。
     免费服务有时候会"睡觉"，醒来需要一点时间。

问题2：上传图片没反应？
答：检查图片是不是太大，就像微信发不了太大的视频一样。
     换一张小一点的图片试试（最好小于5MB）。

答：没关系！这个只是辅助工具，就像计算器一样。

     里面就有网址，就像微信里有聊天记录一样。

问题5：想关掉程序怎么办？
答：不用关！就像电视关了一样，不用的时候它自己会休息。
     免费版本15分钟不用就会自动"睡觉"。

问题6：程序运行很慢？
答：正常！因为是免费的，就像免费WiFi有时候会慢一样。
     耐心等一等，就像等公交车一样。

问题7：出错了，有很多红色字？
答：别着急！就像手机死机了一样，重启一下就好。
     回到Render网站，找到您的项目，点"Restart"按钮。
