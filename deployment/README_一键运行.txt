儿童手腕隐匿性骨折检测系统 - 一键运行指南

=====================================================

重要提示：请确保您的电脑已安装 Python 3.8 或更高版本，并且已连接到互联网以下载依赖包。

⚠️ 编码问题说明：为避免中文乱码，请使用英文启动脚本。

1. 快速启动（推荐）
   - 双击运行 launch_system.bat      # 英文界面，无乱码问题
   - 根据提示等待依赖安装完成（首次运行需要几分钟）
   - 系统将自动启动后端服务并打开前端界面

2. 备用启动方式
   - final_launch_ansi.bat    全自动启动脚本（备用）
   - run_system_ansi.bat      手动打开前端版本
   - start.bat                中文版本（可能显示乱码）
   - start_simple.bat         简单中文版本
   - run.bat                 原始启动脚本

3. 文件说明
   - launch_system.bat          主启动脚本（英文，自动打开前端）
   - run_system_ansi.bat        手动打开前端版本（英文）
   - final_launch_ansi.bat      全自动启动脚本（备用，英文）
   - start.bat                  原始中文启动脚本（可能乱码）
   - backend.py                 后端 FastAPI 服务
   - fracture_frontend.html     前端用户界面（英文文件名）
   - requirements.txt           依赖包列表
   - model/model_3.pth          分类模型文件
   - yolo/yolov8_fracture_best.pt  YOLO检测模型文件（在yolo目录）
   - exit/uploads/              上传图片保存目录（自动创建）

3. 系统要求
   - 操作系统：Windows 7/8/10/11
   - 内存：至少 4GB RAM（建议 8GB）
   - 磁盘空间：至少 2GB 可用空间
   - 网络：首次运行需要联网下载依赖

4. 常见问题
   Q: 启动时提示“未检测到Python”
   A: 请先安装 Python 3.8+，并确保勾选“Add Python to PATH”

   Q: 依赖安装失败
   A: 请检查网络连接，或手动运行：
      pip install -r requirements.txt

   Q: 端口 8000 被占用
   A: 请关闭占用 8000 端口的程序，或等待10秒后重试

   Q: 前端页面无法连接到后端
   A: 确保后端服务已启动（命令行窗口显示服务运行中）

5. 联系方式
   如有问题，请联系项目开发者。
